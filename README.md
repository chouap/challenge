Challenge
===================


Création de la base de données
-------------

Jouer les fichiers sql dans le répertorie "sql" :

- structure.sql
- data.sql

----------


Routes disponible
-------------------

Récupérer un article :

url: article/{id} 
méthode : GET


Récupérer un auteur :

url : autor/{id}
method : GET


Ajouter un article :

url : article
paramètre : title, content, publicationDate, autorId (case sensitive)
method : POST
